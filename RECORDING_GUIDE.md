# دليل تسجيل الصوت لتطبيق عالم بدر

## 1) نطق الكلمات العربية
سجل كل كلمة كملف MP3 قصير وواضح، مثل:
- asad.mp3 = أسد
- fil.mp3 = فيل
- apple_ar.mp3 = تفاح

يمكن لاحقاً ربطها في ملف JSON بدل نطق المتصفح.

## 2) أصوات الحيوانات
ضع أصوات الحيوانات الحقيقية في:
`assets/audio/animals/`

الأسماء المطلوبة حالياً:
- roar.mp3
- trumpet.mp3
- monkey.mp3
- neigh.mp3
- moo.mp3
- baa.mp3
- woof.mp3
- meow.mp3
- cluck.mp3
- quack.mp3

## نصيحة التسجيل
استخدم صيغة MP3، مدة 1 إلى 3 ثوانٍ، صوت واضح بدون ضجيج، وحجم منخفض نسبياً حتى لا يزعج الطفل.
