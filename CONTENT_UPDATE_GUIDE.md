# طريقة تحديث محتوى PWA

1. افتح `assets/data/words.json`.
2. أضف الكلمات داخل مصفوفة `words` بنفس الشكل:

```json
{
  "id": 2001,
  "grade": "3",
  "word_en": "school",
  "meaning_ar": "مدرسة",
  "source": "Jordanian curriculum"
}
```

3. ارفع الملف إلى GitHub.
4. افتح التطبيق واضغط: الإعدادات > تحديث القاموس من JSON.

الصفوف المعتمدة:
- KG
- 1
- 2
- 3
- 4
- 5
