# قاموسي المدرسي PWA - نسخة مدمجة الكلمات

هذه النسخة تحتوي على الكلمات مدمجة مباشرة داخل `app.js`، لذلك يظهر القاموس حتى لو فشل تحميل `assets/data/words.json`.

- تعمل على iPhone وAndroid من المتصفح.
- قابلة للإضافة إلى الشاشة الرئيسية.
- تحتوي 1350 كلمة: 150 كلمة لكل صف من الروضة إلى الثامن.
- زر تحديث JSON يبقى متاحاً للتحديث لاحقاً.
