# عالم بدر للتعلم والمرح - نسخة التصميم المبهر

هذه النسخة تطبق التصميم الجديد المستوحى من الصورة: شاشة رئيسية مبهرة، بدر الدب، بطاقات كبيرة، ألعاب متعددة، وقسم تسجيل صوت عربي بشري.

## أهم الإضافات
- صورة تصميم رئيسية داخل `assets/images/showcase.png`.
- واجهة ألعاب أكبر وأكثر جاذبية.
- صفحة عنصر بتصميم مشهد مصور.
- تسجيل صوت عربي لكل عنصر وقصة عبر الميكروفون.
- تشغيل التسجيل البشري أولاً، وإن لم يوجد يستخدم نطق الجهاز.
- أصوات حيوانات كرتونية موجودة داخل `assets/audio/animals`.

## ملاحظة iPhone
التسجيل يحتاج فتح التطبيق عبر HTTPS أو localhost، والسماح للميكروفون.
